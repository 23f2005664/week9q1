Sample Financial Model (Python)


import numpy as np

def npv(cashflows, rate):
    return sum(cf / (1+rate)**i for i, cf in enumerate(cashflows, start=1))

cash = [-1000, 300, 400, 500]
print('NPV @ 8% =', npv(cash, 0.08))

Financial Formula

Weighted Average Cost of Capital (WACC):

$$\text{WACC}=\frac{E}{V}\times R_e + \frac{D}{V}\times R_d \times (1-T_c)$$

Define each term: E=equity, D=debt, V=E+D, R_e=cost of equity, R_d=cost of debt, T_c=tax rate.
